# calorie-tracker

## Installing

Since moving to react v17 error corrupts npm installs so yarn is required for now

```console
$ cd calorie-tracker
$ npm i
$ cd client_web
$ yarn install 
$ cd ..
$ cd api
$ npm i
```
### /api/.env Example

```console
PORT=5000
DB_URI=
APP_SECRET=shhhhhhhh
```

## Running

```console
$ cd calorie-tracker
$ npm start
```
