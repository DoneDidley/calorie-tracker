# calorie-tracker


## SETUP
---
since moving to react v17 error corrupts npm installs so yarn is required for now
### Installing
```shell
$ cd calorie-tracker
$ npm i
$ cd client_web
$ yarn install 
$ cd ..
$ cd api
$ npm i
```
### /api/.env Example
```shell
PORT=5000
DB_URI=
APP_SECRET=shhhhhhhh
```

### Running
```shell
$ cd calorie-tracker
$ npm start
```
